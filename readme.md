# ![](./static/1_Desktop_Icons/icon_048.png) Evern

> Unofficial Evernote App wrapping Web UI


## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
